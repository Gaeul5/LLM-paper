## LLM 문화적 특성 분석 연구
(Measurement and Analysis of Cultural Characteristics in Large Language Models)

## 연구 개요
- 주제: 호프스테드 문화 차원 이론을 통한 ChatGPT와 Clova X의 답변 편향성 비교
- 주요 내용: 질문 언어(한국어/영어)에 따른 인공지능 모델의 가치관 투영 차이 분석
